## 快速开始

### 开发模式运行

1. 安装 Python 3.14 和依赖：
   ```
   pip install customtkinter Pillow
   ```

2. 双击 `启动压枪控制中心v2.bat`，或命令行运行：
   ```
   python src\recoil_ui_v2.py
   ```

### 打包为EXE

1. 安装 PyInstaller：
   ```
   pip install pyinstaller
   ```

2. 运行打包脚本：
   ```
   scripts\build_pyinstaller.bat
   ```

3. 打包产物在 `dist\pyinstaller\` 目录下

### 打包分发ZIP

```
scripts\打包分发.bat
```

## 核心功能

- **压枪控制**：鼠标左键+右键同时按下触发
- **LUA脚本导入**：支持导入游戏LUA压枪脚本
- **轨迹可视化**：实时显示压枪轨迹
- **力度/倍镜**：可调节压枪力度和倍镜倍率
- **武器管理**：批量选择、删除、启用/禁用
- **武器图片**：支持2K分辨率武器图片显示
- **搜索过滤**：实时搜索武器（防抖优化）

## 技术栈

| 组件 | 技术 |
|------|------|
| UI | customtkinter（暗黑主题） |
| 压枪核心 | IbInputSimulator.dll（驱动级鼠标模拟） |
| 键鼠检测 | GetAsyncKeyState API |
| 配置管理 | JSON |
| 打包 | PyInstaller（推荐）/ Nuitka |

## 注意事项

- **Nuitka兼容性**：Python 3.14 对 Nuitka 只是实验性支持，编译的EXE可能崩溃（0xC000001D），推荐使用 PyInstaller
- **VC++运行时**：打包时需包含 vcruntime140.dll 和 vcruntime140_1.dll
- **管理员权限**：部分游戏需要管理员权限才能正常压枪
