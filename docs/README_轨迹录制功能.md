# 轨迹录制功能使用说明

## 功能简介

轨迹录制功能是从 `recoil_v4.0_优化版.py` 中提取并集成到压枪UI中的功能模块。它允许用户通过可视化界面手动标记弹孔位置，生成压枪轨迹数据。

## 主要功能

### 1. 可视化标记
- 在画布上点击添加标记点
- 拖拽移动标记点位置
- 支持撤销操作
- 显示标记序号和连线

### 2. 数据生成
- 自动生成相对位移数据
- 支持小数精度
- 同时生成 `lua_pattern` 和 `pattern` 格式

### 3. 数据导入导出
- 导入现有的JSON或LUA数据
- 导出为JSON文件
- 直接添加到武器库

## 使用方法

### 打开轨迹录制器

1. 启动压枪控制中心 v2.0
2. 在左侧面板点击 **"轨迹录制"** 按钮（橙色）
3. 打开轨迹录制窗口

### 录制轨迹

1. **设置武器信息**
   - 输入武器名称
   - 设置射速（毫秒）

2. **添加标记点**
   - 在画布上点击左键添加标记
   - 标记点会自动编号
   - 相邻标记点之间会显示连线

3. **编辑标记点**
   - 点击标记点选中它
   - 拖拽移动标记点位置
   - 右键打开菜单进行更多操作

4. **微调标记**
   - 使用列表选择标记点
   - 右键菜单编辑坐标

### 生成数据

1. 点击 **"生成数据"** 按钮
2. 数据会自动添加到武器库
3. 可以在压枪UI中选择使用

### 导入现有数据

1. 点击 **"导入数据"** 按钮
2. 选择JSON数据文件
3. 数据会加载到画布上

### 导出数据

1. 点击 **"导出JSON"** 按钮
2. 选择保存位置
3. 生成标准格式的武器配置文件

## 快捷键

- **Ctrl+Z**: 撤销上一个标记
- **Delete**: 删除选中的标记
- **Escape**: 取消选择

## 数据格式

生成的数据包含两种格式：

### lua_pattern 格式
```json
{
  "lua_pattern": [
    {"x": 0, "y": 0, "d": 10},
    {"x": 1.5, "y": 5.2, "d": 10},
    {"x": -0.8, "y": 6.1, "d": 10}
  ]
}
```

### pattern 格式
```json
{
  "pattern": [0, 5, 6, 7, 8]
}
```

## 注意事项

1. **坐标系统**
   - 原点在画布中心
   - X轴向右为正
   - Y轴向下为正

2. **数据精度**
   - 支持小数坐标
   - 相对位移保留2位小数

3. **兼容性**
   - 生成的数据兼容现有压枪系统
   - 支持力度控制和倍镜切换

## 示例工作流程

### 从图片录制

1. 在游戏中截图弹孔分布
2. 打开轨迹录制器
3. 根据弹孔位置点击标记
4. 生成数据并添加到武器库

### 从现有数据导入

1. 点击"导入数据"
2. 选择LUA或JSON文件
3. 在画布上查看轨迹
4. 可以编辑后重新导出

## 技术实现

- 使用 tkinter Canvas 绘制界面
- 支持缩放和平移（未来版本）
- 数据格式兼容 `recoil_v4.0_优化版.py`
