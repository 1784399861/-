# 压枪控制中心 - 现代化UI版本

## 概述

压枪控制中心是一个基于 IbInputSimulator 库的现代化压枪软件，提供直观的图形用户界面，支持武器管理、参数设置、日志查询和联网配置。

## 功能特性

### 🎯 核心功能
- **武器管理**：选择、新增、删除武器配置
- **参数设置**：调整压枪模式、射速、启用状态等
- **日志查询**：实时查看运行日志，支持导出
- **联网配置**：同步武器配置，检查更新

### 🎨 界面特点
- **现代化UI**：基于 customtkinter 的暗黑主题
- **响应式布局**：左侧武器列表，右侧参数设置
- **实时状态**：显示当前武器和压枪状态
- **快捷操作**：一键启用/禁用压枪功能

## 安装与运行

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行程序
- **方法一**：双击 `启动压枪控制中心.bat`
- **方法二**：命令行运行 `python recoil_ui.py`

### 3. 管理员权限
某些游戏需要管理员权限才能生效，程序会自动检测并提示。

## 界面说明

### 左侧面板 - 武器库
- **武器列表**：显示所有已配置的武器
- **新增武器**：点击"新增武器"按钮添加新武器
- **删除武器**：选择武器后点击"删除武器"按钮
- **选择武器**：点击武器项即可选择

### 右侧面板 - 参数设置
#### 武器参数选项卡
- **武器名称**：修改武器名称
- **武器描述**：添加武器描述信息
- **压枪模式**：设置每发子弹的下移像素（逗号分隔）
- **射速**：设置射击间隔（毫秒）
- **启用状态**：启用或禁用此武器

#### 全局设置选项卡
- **开关热键**：设置压枪开关热键（默认F1）
- **开火键**：设置开火按键（默认鼠标左键）
- **移动间隔**：设置鼠标移动间隔（毫秒）
- **开机启动**：设置是否开机自动启动

#### 网络配置选项卡
- **配置服务器**：设置武器配置同步服务器
- **同步武器配置**：从服务器同步最新武器配置
- **检查更新**：检查软件更新

### 底部面板 - 运行日志
- **实时日志**：显示程序运行日志
- **清空日志**：清空当前日志显示
- **导出日志**：将日志导出为文本文件

## 配置文件

### 武器配置 (config/weapons.json)
```json
{
  "武器名称": {
    "name": "显示名称",
    "pattern": [2, 3, 4, 5, 6, 7, 8, 9, 10],
    "fire_rate": 10,
    "enabled": true,
    "description": "武器描述"
  }
}
```

### 全局设置 (config/settings.json)
```json
{
  "hotkey": "F1",
  "fire_button": "LButton",
  "move_interval": 10,
  "auto_start": false,
  "log_level": "INFO"
}
```

## 使用流程

### 1. 选择武器
1. 在左侧武器列表中点击要使用的武器
2. 右侧会显示该武器的详细参数
3. 状态栏会显示当前选中的武器

### 2. 调整参数
1. 在"武器参数"选项卡中修改参数
2. 点击"保存武器参数"按钮保存
3. 参数会立即生效

### 3. 启用压枪
1. 点击右上角的"启用压枪"开关
2. 状态栏会显示"已启用"
3. 按下开火键时会自动压枪

### 4. 查看日志
1. 底部面板显示实时运行日志
2. 可以点击"导出日志"保存日志
3. 可以点击"清空日志"清除显示

## 压枪模式说明

压枪模式是一个数组，每个元素代表对应子弹的下移像素值。

### 示例模式
- **稳定模式**：`[5, 5, 5, 5, 5, 5, 5, 5, 5]` - 固定下移量
- **递增模式**：`[2, 3, 4, 5, 6, 7, 8, 9, 10]` - 逐渐增加下移量
- **脉冲模式**：`[8, 8, 8, 0, 8, 8, 8, 0, 8, 8, 8]` - 模拟点射

### 调整建议
1. **新手**：从稳定模式开始，使用固定值
2. **进阶**：根据武器特性调整递增模式
3. **高级**：使用脉冲模式模拟点射

## 常见问题

### 1. 程序无法启动
- 检查是否安装了 Python 3.x
- 运行 `pip install customtkinter` 安装依赖
- 确保 IbInputSimulator.dll 在程序目录

### 2. 压枪无效
- 检查是否以管理员权限运行
- 确认已启用压枪功能
- 检查武器是否已启用

### 3. 日志不显示
- 检查 config 目录是否存在
- 检查是否有写入权限
- 重启程序

### 4. 联网功能无法使用
- 检查网络连接
- 确认服务器地址正确
- 检查防火墙设置
