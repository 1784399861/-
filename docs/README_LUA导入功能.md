# LUA压枪脚本导入功能说明

## 功能概述

本次更新为压枪控制中心添加了LUA压枪脚本导入功能，支持从罗技G系列鼠标宏脚本中导入压枪数据。

## 新增功能

### 1. LUA解析器 (`lua_parser.py`)
- 解析LUA格式的压枪脚本
- 提取WEAPON_DATA表中的武器数据
- 支持42种武器，每种武器包含30-50个压枪点
- 解析倍率配置和基础灵敏度参数

### 2. LUA导入功能
- 在武器面板添加"导入LUA"按钮
- 支持文件选择对话框
- 自动导入所有武器到配置
- 显示导入结果和配置信息

### 3. 压枪轨迹可视化
- 在武器参数选项卡添加Canvas可视化区域
- 支持显示lua_pattern的x, y坐标轨迹
- 支持显示简单Y轴压枪模式柱状图
- 实时更新轨迹预览

### 4. 力度控制
- 添加力度控制滑块（0.1x - 3.0x）
- 保存力度值到武器配置
- 在压枪核心中应用力度乘数

### 5. 倍镜切换
- 添加倍镜切换下拉菜单（1.0x - 6.0x）
- 保存倍镜值到武器配置
- 在压枪核心中应用倍镜乘数

## 使用方法

### 导入LUA脚本
1. 启动压枪控制中心 v2.0
2. 点击左侧面板的"导入LUA"按钮
3. 选择LUA压枪脚本文件（如`新脚本 V4.10.lua`）
4. 等待导入完成，查看导入结果
5. 在武器列表中查看导入的武器

### 查看压枪轨迹
1. 在武器列表中选择一个武器
2. 在右侧参数设置面板中查看"压枪轨迹预览"
3. 轨迹图显示压枪的x, y坐标移动

### 调整力度控制
1. 在武器列表中选择一个武器
2. 在右侧参数设置面板中找到"力度控制"滑块
3. 拖动滑块调整力度（0.1x - 3.0x）
4. 点击"保存武器参数"保存设置

### 切换倍镜
1. 在武器列表中选择一个武器
2. 在右侧参数设置面板中找到"倍镜切换"下拉菜单
3. 选择倍镜倍率（1.0x - 6.0x）
4. 点击"保存武器参数"保存设置

## 技术细节

### LUA脚本格式
```lua
local WEAPON_DATA = {
{
    name = "AKM_XY",
    pattern = {{x=0,y=0,d=3.5},{x=1.08,y=5.16,d=20.0}, ...},
    key = 0,
    key_ctrl = KEY_AKM_XY
},
...
}
```

### 解析器输出格式
```json
{
  "AKM_XY": {
    "name": "AKM_XY",
    "pattern": [0, 5, 5, 6, ...],
    "fire_rate": 20,
    "enabled": true,
    "description": "从LUA导入: AKM_XY",
    "lua_pattern": [
      {"x": 0.0, "y": 0.0, "d": 3.5},
      {"x": 1.08, "y": 5.16, "d": 20.0},
      ...
    ],
    "lua_key": 0,
    "lua_key_ctrl": 0,
    "strength": 1.0,
    "scope": "1.0x"
  }
}
```

### 压枪核心修改
- 修改`_execute_recoil_pattern`方法
- 应用力度控制和倍镜切换乘数
- 计算总乘数：`total_multiplier = strength * scope_multiplier`
- 应用到鼠标移动量：`move_y = int(move_y * total_multiplier)`

## 注意事项

1. LUA脚本需要是罗技G系列鼠标宏格式
2. 导入的武器会自动添加到武器列表
3. 力度控制和倍镜切换会影响压枪效果
4. 建议根据游戏实际情况调整参数
5. 某些游戏可能需要管理员权限

## 故障排除

### 导入失败
- 检查LUA文件格式是否正确
- 确认LUA文件包含WEAPON_DATA表
- 查看日志了解错误信息

### 轨迹不显示
- 确认武器有压枪数据
- 检查Canvas是否正确初始化
- 尝试重新选择武器

### 压枪效果不理想
- 调整力度控制滑块
- 切换倍镜倍率
- 根据游戏调整压枪模式
